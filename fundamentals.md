# Linux User Management - Fundamentals

## Users
Users are accounts that can log into a Linux system and perform tasks based on permissions.

## Groups
Groups allow multiple users to share access to files and directories.

## Key Commands

- useradd → Create a new user
- passwd → Set or change password
- usermod → Modify user (add to group)
- groupadd → Create group
- chage → Manage password expiry
- sudo / visudo → Provide controlled admin access

## Concepts

### Primary vs Secondary Group
- Primary group: default group of a user
- Secondary group: additional groups for access

### Principle of Least Privilege
Users should only have access required for their role.
